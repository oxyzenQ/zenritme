# zenritme — a quiet terminal timer

Minimal terminal timer, stopwatch, and pomodoro.

## Usage

```
zenritme --timer-up
zenritme --timer-down <DURATION>
zenritme --stopwatch
zenritme --pomodoro [FOCUS BREAK]
zenritme --help
```

Duration format: `30s`, `10m`, `1h`

### Controls

| Key | Action |
|-----|--------|
| `q` / `Esc` | Quit |
| `p` | Pause / resume |
| `r` | Reset current session |
