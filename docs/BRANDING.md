# Zenritme Brand Guidelines

This document defines the visual identity and communication standards for the Zenritme project. It ensures consistent branding across all touchpoints — from the GitHub repository to the CLI output and documentation.

---

## 1. Brand Identity

**Zenritme** is a terminal-based Pomodoro timer designed for focus and productivity. The brand reflects:

- **Focus** — minimal, distraction-free interface
- **Rhythm** — structured work/break cycles
- **Zen** — calm, aesthetic terminal visuals

---

## 2. Name Usage

### 2.1. Correct forms

| Context | Format |
|---|---|
| Running text / prose | Zenritme |
| Titles / headings | Zenritme |
| Code / CLI | `zenritme` (lowercase) |

### 2.2. Incorrect forms

- ~~ZenRitme~~ (no internal capitalization)
- ~~ZenritmeX~~ (derivative form)

---

## 3. Logo

### 3.1. Logo file

The official logo is located at [`assets/zenritme-new-logo.png`](../assets/zenritme-new-logo.png).

### 3.2. Usage rules

- **Clear space**: maintain padding equal to at least 25% of the logo height on all sides
- **Aspect ratio**: always preserve the original aspect ratio — do not stretch or distort

---

## 4. Tone of Voice

Zenritme's communication should be calm, helpful, and focused.

- **Minimalist** — keep documentation and CLI output concise
- **Encouraging** — use positive tone for productivity-related messaging
- **Professional** — maintain technical accuracy for installation and configuration

---

## 5. Third-party Usage

External projects or articles referencing Zenritme should:

- Use the correct project name: Zenritme
- Link to the official repository: <https://github.com/oxyzenQ/zenritme>
